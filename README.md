## Install

This project uses [node](https://nodejs.org) and [npm](https://www.npmjs.com). 

```sh
$ npm install nitpicky-cli
$ # OR
$ yarn add nitpicky-cli
```

## Usage

In root of project:

```sh
$ nitpick
# or
$ nitpick [file|dir|glob]
```
